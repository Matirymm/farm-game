# 🌾 مزرعتي — Farm Game (Flutter + Flame)

لعبة محاكاة مزرعة — المرحلة 1: هيكل قابل للبناء والتشغيل **دون أي أدوات محلية**.

## كيف تحصل على APK يعمل على جوالك (بدون تثبيت أي شيء)

1. **ارفع هذا المجلد إلى مستودع GitHub جديد** (مثلاً `Matirymm/farm-game`)، الفرع `main`.
2. سجّل في **codemagic.io** بحساب GitHub (مجاني، 500 دقيقة/شهر، بدون بطاقة).
3. أضف التطبيق → اختر المستودع → سيكتشف `codemagic.yaml` تلقائياً → **Start new build**.
4. بعد ~8–12 دقيقة: حمّل `app-release.apk` من صفحة البناء وثبّته على جوال أندرويد.

> السرّ: خطوة `flutter create .` في `codemagic.yaml` تولّد مجلد `android/` كاملاً
> في السحابة — لهذا المستودع لا يحتاج سقالة أندرويد ولا Android Studio.

## بنية المشروع

```
lib/
├── main.dart          # نقطة الدخول + ربط الـ Overlays
├── economy.dart       # كل أرقام التوازن (المصدر الوحيد للحقيقة)
├── state.dart         # حالة اللعبة + حفظ محلي + نمو أثناء الإغلاق
├── game/
│   ├── farm_game.dart # مشهد Flame: خريطة، مبانٍ، قطع، حيوانات
│   ├── plot.dart      # قطعة أرض: زراعة/نمو متحرك/حصاد/توسعة
│   └── animal.dart    # حيوان متجول: مشي متمايل + إطعام/جمع
└── ui/
    ├── hud.dart       # عملات/ماس/XP + السوق (بيع، شراء حيوانات، ترقية مستودع)
    └── seed_menu.dart # قائمة اختيار البذور
```

## طريقة اللعب (النسخة الحالية)

- المس قطعة فارغة → اختر بذوراً → المحصول ينمو ويتمايل → المسه عند الجاهزية للحصاد.
- المس محصولاً أثناء النمو → تسريع فوري مقابل 💎1.
- زر «السوق والحظيرة» → بيع المخزون، شراء حيوانات، ترقية المستودع.
- المس الحيوان: 🍽️ إطعام (يستهلك محاصيل من مخزونك) → ⏳ إنتاج → ❗ جمع.
- النمو يستمر أثناء إغلاق التطبيق (مؤقتات epoch حقيقية).

## استبدال الإيموجي بأصول فنية حقيقية

الرسومات الحالية إيموجي عمداً — كل عنصر مرسوم عبر `TextComponent` واحد،
فاستبداله بـ `SpriteComponent` تغيير موضعي لا يمس منطق اللعبة:

1. حمّل حزمة **Sprout Lands** (Cup Nooble على itch.io — مجانية/بالتبرع، بكسل آرت
   مزرعة كاملة: محاصيل بمراحل نمو، دجاج، أبقار، مبانٍ) أو حزم **Kenney.nl** (CC0).
2. ضع ملفات PNG في `assets/images/`.
3. في `plot.dart` و`animal.dart`: استبدل `TextComponent` بـ
   `SpriteAnimationComponent` (مراحل النمو = إطارات، المشي = أنيميشن).

## ملاحظات توازن

- الأوقات حالياً **بالثواني** للاختبار. قبل الإطلاق: عدّل `growSeconds` و
  `produceSeconds` في `economy.dart` إلى القيم الحقيقية (×60).
- كل الأرقام في ملف واحد (`economy.dart`) تمهيداً لنقلها إلى
  Firebase Remote Config في المرحلة 4.
